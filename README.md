# Principios SOLID (Java)

Repositorio de ejemplos en Java para los 5 principios SOLID:

- **S**RP (Single Responsibility Principle) → `srp/`
- **O**CP (Open/Closed Principle) → `ocp/`
- **L**SP (Liskov Substitution Principle) → `lsp/`
- **I**SP (Interface Segregation Principle) → `isp/`
- **D**IP (Dependency Inversion Principle) → `dip/`

Cada carpeta incluye:
- un ejemplo “mal” (violación del principio),
- un ejemplo “bien” (refactor),
- una explicación corta.
